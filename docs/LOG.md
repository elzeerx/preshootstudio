- Surface: `#FFFFFF` (HSL: 0 0% 100%)
- Text Main: `#0B1120` (HSL: 220 26% 5%)
- Text Secondary: `#6B7280` (HSL: 220 9% 46%)

**المكونات الأساسية**:
- Button مع variants (primary, secondary, outline, ghost)
- Card component
- Typography helpers (heading-1, heading-2, heading-3, body-text)
